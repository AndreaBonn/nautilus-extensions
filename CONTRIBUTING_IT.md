# Contribuire alle Estensioni Nautilus

Grazie per il tuo interesse nel contribuire! 🎉

## Come Contribuire

### Segnalare Bug

Se trovi un bug, apri una issue con:
- Una descrizione chiara del problema
- Passaggi per riprodurlo
- Comportamento atteso vs comportamento effettivo
- Informazioni sul tuo sistema (OS, versione Nautilus, versione Python)

### Suggerire Funzionalità

I suggerimenti sono benvenuti! Apri una issue descrivendo:
- La funzionalità che vorresti vedere
- Perché sarebbe utile
- Come potrebbe funzionare

### Inviare Codice

1. Fai un fork del repository
2. Crea un nuovo branch (`git checkout -b feature/tua-funzionalita`)
3. Fai le tue modifiche
4. Testa accuratamente le modifiche
5. Fai commit con messaggi chiari (`git commit -m 'Aggiungi funzionalità X'`)
6. Pusha sul tuo fork (`git push origin feature/tua-funzionalita`)
7. Apri una Pull Request

### Stile del Codice

- Segui PEP 8 per il codice Python
- Usa nomi di variabili e funzioni significativi
- Aggiungi commenti per la logica complessa
- Mantieni le funzioni focalizzate e piccole
- Testa la tua estensione prima di inviarla

### Testing

Prima di inviare:
1. Testa l'estensione con vari tipi e dimensioni di file
2. Verifica che funzioni su un'installazione pulita di Nautilus
3. Controlla memory leak con file grandi
4. Assicurati che la gestione degli errori funzioni correttamente

## Domande?

Sentiti libero di aprire una issue per qualsiasi domanda!
